# 🏦 Loan Eligibility Predictor

An AI-powered full-stack web app that predicts loan eligibility with explainability.

## 🚀 Features
- ✅ XGBoost ML model with 85%+ accuracy
- 🔬 SHAP explainability — shows WHY a loan is approved/rejected
- 🎛️ What-If simulator — change income/loan and see impact live
- 🏦 Multi-bank comparison (SBI, HDFC, ICICI, Axis)
- 📄 Downloadable loan report
- 🌐 Full REST API with Swagger docs

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| ML Model | XGBoost + SHAP |
| Backend | FastAPI + Python |
| Frontend | React + Vite + Recharts |
| Deployment | Render + Vercel |
